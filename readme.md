## DSA Practice Repository

This repository is used to build consistency in solving coding challenges and improving problem-solving skills across different languages.

---

## Structure

```
dsa/
├── typescript/        # TypeScript-based challenges
│   └── leetcode/
│
├── python/        # Python exercises and practice
│   └── w3schools-dsa/
│
├── .gitignore
└── README.md
```

---

## TypeScript

Contains coding challenges solved using TypeScript.

### Setup

```bash
cd typescript
npm install
```

### Running a file

```bash
npx tsx <folder-name>/<file-name>.ts
```

---

## Python

Contains Python exercises and practice problems.

### Running a file

```bash
cd python
python <folder-name>/<file-name>.py
```

---

## Goal

* Practice Data Structures and Algorithms regularly
* Improve problem-solving skills
* Work across multiple languages (TypeScript and Python)
